Binary Switcher

This is just a small Constructor to create Binary Switchers, binary strings (rendered as numbers) that records a set of boolean values that will be assigned to labeled propierties... pretty much like having an object to store boolean keys only... but in binary strings, and stuff.

Why? You should be asking, well... because I can. Also, I want to practice Binary Operators... and this is the first stuff that came into my mind.
